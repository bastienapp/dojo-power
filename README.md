## Puissance

Créer une méthode récursive `pow` qui calcule une puissance de `a` par `b` :

	2⁰ = 1
    2¹ = 2
    2² = 4
    2³ = 8
    2⁴ = 16
    2⁵ = 32

Rappel des commande junit :

    javac -cp .:junit-4.12.jar PowerTest.java
    java -cp .:junit-4.12.jar:hamcrest-core-1.3.jar org.junit.runner.JUnitCore PowerTest
